BDD2 :
Ce dossier contient tous les fichiers nécessaire au lancement de notre programme sous Eclipse.
BDD2/libs contient les différentes librairies java que nous utilisons, veillez à ce que votre IDE les retrouve
afin que notre programme se lance correctement.
BDD2/src contient tous les .java de notre programme.

Nous avons créé un utilisateur admin, avec pour mot de passe admin, dans notre BDD, cet utilisateur correspond à l'application 
qui se connecte à la BDD, il a donc seulement les droits que nécessite l'application. 
Si vous voulez lancer notre programme sans modifier notre code vous devez vous aussi créer cet utilisateur.

SCRIPT :
Ce .sql contient le script SQL que nous utilisons pour créer tous les tableaux de notre base de donnée.

RAPPORT :
Ce pdf contient notre rapport.
